# Killed candidates

Dead hypotheses, kept on purpose. This directory is the trial-count
ledger that keeps the successes honest: anyone who wants to claim the
framework only publishes its hits can count the misses here.

Format: one directory per killed candidate, moved here from
`candidates/` with its CLAIM.md (pre-registration), derivation, and
post-mortem intact.

## Precedent (from the repo's own history, pre-dating this directory)

**H1: r = 5·(α/2π) for the UV gravity residual.** The lead that
motivated a pre-registered family scan. Deviation 7.8e-4, but the
trial-corrected expectation was ~11 false positives: indistinguishable
from a look-elsewhere accident. KILLED 2026-06-11 and retracted. The
same scan's H2 lead (the 9/13 face share) later survived because it
came back two orders of magnitude sharper than its siblings and was
subsequently derived (the face-split law, now canonical). That pair is
the standard this directory enforces: the same scan produced one kill
and one promotion, and both are recorded.
